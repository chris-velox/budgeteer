# Budgeteer

Budgeteer is a web application that provides
- Transaction tracking for
    - Savings and checking accounts
    - Credit card accounts
    - Liabilites
    - Assets
    - Investments
- Multiple budgeting
    - "Turn-based" (Civ) budgeting
    - Envelop budgeting
- Reporting
    - Tables/matrices
    - Visualizations
